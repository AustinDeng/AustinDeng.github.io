## 威科夫操盘法的定义

威科夫操盘法（Wyckoff Method）是20世纪初由美国技术分析先驱理查德·D·威科夫创立的一套完整交易体系。它通过分析价格与成交量的关系，识别市场主力（称为“综合人”）的吸筹、拉升、派发、下跌等操作循环，从而跟随主力意图捕捉交易机会。

## 关键知识点

垂直供应柱顶部形成阻力区(供应区)

垂直需求柱底部形成支撑区，需要画线
`如果支撑区触底后没有强烈反弹，支撑失效`
`支撑挺住：小蜡烛 -> 供应枯竭 -> 吸引需求`

支撑是否有效，主要看从支撑处起来的强度

冰线：新高前反弹波段的低点，或下降趋势中回调前的反弹低点

突破不能进场，突破时公众行为，不能与公众为伍
此时安全的进场点时：价格以小蜡烛的低量回调时(供应缺乏，需求主导)

超买线与水平阻力线交叉，形成死角

震荡区需求扩大：
1. 在震荡区右手边
2. 相对长的阳线，包括下影线
3. 扩大的成交量

熊市中价格下跌不需要有扩大的成交量：**因为买盘数量少，若成交量扩大，是买盘扩大导致的**

恐慌抛售：天量配上超长阴线，不是市场大跌的信号，而是市场遇到巨大买盘的信号

熊市中的放量滞跌和牛市中的放量滞涨都属于停止行为

牛市中，不希望看到超长的阳线和放量，**因为这不是真正的牛市行情，而是CM(主力)的一个出货手段**

因果关系原则：
`因 -> 趋势出现前的准备过程`
`果 -> 趋势形成`

因果关系  -> **1. 不要着急盲目抄底，熊市抄底是大忌！！ 2. 趋势反转之前，必须看到一段震荡区间(吸筹阶段)**

牛市的开始需要准备过程，不能看到大跌就抄底

熊市转牛市的理论基础：
`1. 市场上的供应开始枯竭`
`2. 需求必须扩大并保持力度`

![主力吸筹到派发的过程](/assets/posts/威科夫操盘法/主力吸筹到派发的过程.png)

牛转熊的理论基础：**供应增强，经过派发阶段(区间交易)后，需求耗尽，供应主导市场**


-----


熊市开始：
1. 当日最低价和最高价分别低于昨日最低价和最高价
2. 当日收盘价低于昨日收盘价
3. 当日成交量大于等于昨日成交量

熊市的终止理论基础：**需求扩大后供应耗尽**

从量价行为上要经历四个市场行为：
1. 初次支撑
2. 恐慌抛售
3. 自动反弹
4. 二次测试

其中第三步的自动反弹很可能是由于空头回补导致的
我们希望在这个阶段看到小碎布下跌并伴随成交量递减的现象

不能在这种(自动反弹)来自熊市的第一次反弹进场，因为没有看到二次测试

恐慌抛售后，趋势也可能出现窄幅横盘行为，这种行为会导致熊市继续

**成功的二次测试：价格波动和成交量与恐慌抛售相比都非常小**

停止行为：
1. 下跌过程中，比较大的下影线+扩大成交量；上升过程中，比较大的上影线+扩大的成交量
2. 下跌中，蜡烛大幅缩小，但成交量大幅增加


![识别底部结构](/assets/posts/威科夫操盘法/识别底部结构.png)

终极震仓和弹簧效应后，会出现二次测试，这个二次测试是最低风险的抄底点


![熊转牛的过程](/assets/posts/威科夫操盘法/熊转牛的过程.png)

牛市开始
1. 强势出现(SOS)+测试
2. 跳离震荡区(JOC)+测试

供应的出现，是抄底者的离场信号

正常牛市是匀速上涨的，不会有急升的现象

上升趋势中的上冲回落属于探索，不是反转信号

需求控制的背景下，回调停在半路是牛市的特征

终极震仓和SOS(包括JOC)是确认牛市的重要行为，**在底部未见其中之一，不要轻易抄底**


------

CM(主力机构)出货过程：
1. 牛市结束的过程
    - 初次供应
    - 抢购高潮 --> 天量新高
    - 自然回落
    - 二次测试
        - 熊市初现SOW
        - 上冲回落
2. 设立出货区，引诱公众买入
3. 测试，确定需求已经耗尽
4. 熊市(SOW)出现

上冲回落：
价格试图突破震荡区顶部，结果没有守住，然后立刻跌回震荡区
**上冲回落必须有跟随才能证明市场供应强劲**

SOW的特点是放量长阴，必须出现在震荡区右手边才生效，**并且紧接无量上涨才能确认**

判断震荡区是吸筹还是派发，关键是测试阶段的市场行为：
1. **如果出现SOS，终极震仓或是弹簧效应 --> 吸筹 --> 牛市恢复**
2. **如果出现SOW，或是带量上冲回落 --> 再派发 --> 熊市**


-----

应该关注的`Spring`在牛市回调的过程中，**千万不要在熊市中找Spring，熊市的Spring是陷阱**


熊市中不能用Spring作为进场工具，抢反弹是公众行为

Spring的重要前提是支撑必须是前面已经建立的强支撑


------

市场本质交易法：分析走势时，先考虑理论基础，然后在图中找能确认理论基础的量价

**熊市不能随意抄底，这是铁律**

空头回补导致的上涨之后，价格迅速带量回落
需求扩大导致的上涨，随后的二次测试过程比较缓慢，成交量以及K线的高低差都相对减小

判断要基于市场倾向，而不是某个形态或指标

牛市中震仓的特点：受消息影响大幅下降，但是没有跟随，确认震仓的是随后的迅速反弹
**如果牛市中出现大幅下跌后没有出现快速反弹，而是出现震荡区，这是派发**

上冲回落(UT)的作用是CM引诱公众追高，买盘跟随确认了UT

确定区间属于吸筹的重要特征：向上JOC或向下震仓(然后迅速上扬)

阻力位置出现停止行为，会导致价格回落

卖盘背景下，Spring经常失败(做空信号)

----

若大阳线伴随成交量大幅增加，但是收盘在中低位，则需要警惕庄家高位出货，涨势或为掩饰撤退，市场疲弱信号显现

下跌K线成交量放大，收盘价偏高，表明庄家在借恐慌吸筹

通常情况下：强势信号出现在阴线中，弱势信号则在阳线中更为常见

压力测试使用原则：如果市场缺乏供应，接下来便需要有“三高量增”的确认，否则这场测试毫无意义

三高：
1. 突破前高
2. 更高的最低价
3. 更高的收盘价
量增：成交量增加

压力测试成功促成上涨趋势的前提条件：
1. 发生在需求强势背景下，或价格跌至前期高成交量区间附近
2. 小下影线，收盘价高于中间价格
3. 下影线最低点低于前一天
4. 成交量较低，低于前两天
5. 两天内必须出现“三高量增”

当压力测试的最低价被突破后，代表的强势行为会被推翻

如果背景中有剧烈的震仓，低量上涨不是无需求，而是在这些价位下几乎没有股票供应

震仓：价格会迅速反弹
破冰：反弹缓慢，需求不足

低成交量的震仓：供应不足，需求主导
高成交量的震仓：需求强烈，市场力量转换

在支撑位上，如果价格触及后出现反转行为，表明机构正在接盘，支撑位真实有效

反转必须有巨量

底部反转后价格出现回落，但成交量明显减少，这是对应底部反转信号的确认，表明市场供应减少

任何量价行为分析都应该放在市场趋势的大背景下

上冲回落(UT)通常发生在阻力位或者横盘区间的顶部

UT突破时：
- 成交量较小：需求不足，价格上涨缺乏机构支撑，后续“三低量增”确认下降趋势
- 成交量大：需求依然保持力度，若做空，需要等二次测试

在上涨趋势的回调中，Spring是一个重要的进场信号

在熊市中的Spring不能作为进场依据

成功的Spring必须伴随“三高量增”，若反而“破冰”(跌破Spring低点)，则表明Spring无法带来上涨趋势

对于高成交量的Spring，一定要等二次测试(低量+小幅度突破)后才行动

UT之后三种特殊情况以及应对策略：
1. 价格下跌，但伴随低量小柱或压力测试
    - 意味市场供应不足，否定看空的判断，价格可能反弹上涨
2. 价格上涨，但伴随无需求特征(成交量小)
    - 确认了UT的有效性，加强看空信号
3. 价格上涨，伴随增量的上影线
    - 意味着机构在更高区间派发，不宜看多


Spring用在支撑上：
- “越过小溪”类型支撑
- 区间底部的支撑
- 回调中的横盘支撑

-----

需求不足：必须是涨价柱，且收盘不在顶部，成交量很低
供应不足：必须是降价柱，且收盘不在底部，成交量很低

在吸筹初期就暴涨，并不代表机构在吸筹，而是明显的空头平仓，后续价格会回落

底部重要量价行为：
1. 接盘行为
    - 恐慌抛售和立刻反弹
    - 压缩行为：巨量不跌
    - 量能停滞行为

2. 确认行为
    - 供应不足：价格再次下跌的时候成交量小，下跌幅度小
    - 高支撑

3. 吸筹手段
    - 疲劳战术
    - 震仓
    - 通过封顶继续低价吸筹
    - 制造做空形态吸引公众做空，然后机构吸筹

4. 进入牛市行为
    - 越过小溪

5. 临界行为


支撑与阻力：
- 区间底部是支撑：强力突破后可能转为阻力(破冰)
- 区间顶部是阻力：强力突破后可能转为支撑(越过小溪)
- 上涨趋势回调50%位是支撑
- 下降趋势反弹50%是阻力
- 超长阳线/下影线的底部是支撑
- 超长阴线/上影线的顶部是阻力

<!-- 当天价格暴跌且成交量大，通常为机构抛售 -->
底部的暴跌多由散户恐慌抛售引发

长期下跌趋势后，暴跌通常为正面信号

上涨趋势中的暴跌
- 迅速反弹或保持平静甚至创新高，则为震仓，上涨延续
- 低价位横盘，趋势可能打破，价格下跌

熊市中的暴涨通常是空头平仓的结果

失败的二次测试也是一个重要的变盘信号

在判断市场顶部时，需要在上涨趋势中观察到两次巨量出货行为

强势行为：上涨有量，下跌无量，或下跌有放量但幅度小
弱势行为：下跌有量，上涨无量，或上涨有放量但幅度小

为确保长期生存和盈利，必须控制每次交易的最大亏损
